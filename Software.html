MATLAB and Simulink

Skill level: Experienced

Some project examples:

MATLAB was introduced to me during University with our first project to simulate projectile motion of a parcel with variable drag induced by a parachute deployment. The tasks stepped up a notch with a project to model the temperature variation across a space shuttle tile using partial derivatives and heat transfer analysis.

MATLAB was the chosen environment for the treasure hunting robot project. Custom search and mark algorithms were used to control an X-Y gantry mounted robot to find and mark magnets hidden beneath sand.

I also used the software to recursively analyse 100+ acoustic waveforms by identifying multiple “click” noises/ peaks in a sample using a threshold-based algorithm.

Simulink was the program of choice for controlling a rolling robot. I used it with an Arduino MKR1000 to interface with a JeVois machine vision camera where it would receive position data and filter out the noise and use a zero-order-hold.

Relevant Projects: Heat Transfer Simulation, Treasure Hunting Robot

CAD software

Skill level: Experienced

Some project examples:

CAD was used for every test rig and University project I have worked on that involve any form of mechanical prototyping. I am accustomed to the iterative design nature of rapid prototyping and designing for assembly with rapid prototyping. This includes the front face of the UV curing chamber which has no visible mechanical fastenings.

When it comes to complicated CAD design, I prefer to use the “Master Geometry” method. This involves creating a part with sketches of key features (datum surfaces, rotation centres/ sweeps, mounting planes) then importing this part into a new part and using the key-features to design and dimension the new part. All geometry is based on a single file making changing dimensions much faster.

Relevant Projects: Card Game Box, UV Curing Chamber: CAD, 4-Point Bending Jig
