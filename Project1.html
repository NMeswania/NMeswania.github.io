My Website

The Brief

I wanted a way to show some of my projects over the years. A paper portfolio would be irritating to lug around and not many people would be able to see unless I showed them in person. So I decided a website would be the best, I could just send a hyperlink and people can peruse at their leisure!

Challenges

I had never tried any web development before, so the key challenge was learning how to develop for the web: coding in HTML, CSS and JS. After laying some initial framework, I signed up to an online web development course to learn more about it.

Developments

I decided to do the frame work first: using “lorem ipsum …” and random images to try and see how things looked and functioned compared to other portfolio-style websites. Random images were, of course, pictures of cute animals as you can see.

Final Design

The final design is what you see here! (It's constantly evolving, so next time you visit, it could look entirely different!).

Further Work

Like a lot of projects, it’s never really finished, I’m still working on this website - making the layout responsive for mobiles/ tablets.

Skills

I learnt how to code in HTML, CSS and JS. I learnt a lot about how the internet works generally too, always useful.
